# Atlas Mod Guide

## What is Atlas?

Atlas is a modular mod suite for StarMade servers. Each feature is packaged as a separate
sub-mod that server administrators can load or unload independently.

## Installed Modules

| Module | Description |
|---|---|
| **AtlasCore** | Core infrastructure — always required |
| **AtlasGuide** | This guide viewer |
| **AtlasBanking** | Player banking and prize bar economy |
| **AtlasBuildSectors** | Private creative building sectors |
| **AtlasExchange** | Blueprint and item marketplace |

## Getting Help

- Use `/guide` to re-open this menu at any time.
- Join the Discord server via the **DISCORD** button in the top-right player menu.

## Banking

If AtlasBanking is installed, you can access your account via the **BANKING** button
in the top-right player menu. From there you can:

- Check your credit balance
- Deposit or withdraw credits
- Transfer credits to other players
- View your transaction history
- Redeem prize bars

## Build Sectors

If AtlasBuildSectors is installed, you have access to a private creative sector via the
**BUILD SECTOR** button. Build sectors are protected from other players and hostile NPCs.
You can invite friends, manage permissions, and spawn blueprints in your sector.

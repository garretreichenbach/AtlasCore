# Atlas Mod Guide

## What is Atlas?

Atlas is a modular mod suite for StarMade servers. Each feature is packaged as a separate
sub-mod that server administrators can load or unload independently.

## Installed Modules

| Module | Description |
|---|---|
| **AtlasCore** | Core infrastructure — always required |
| **AtlasBanking** | Player banking and credit transfers |
| **AtlasBuildSectors** | Private creative building sectors |
| **AtlasExchange** | Blueprint and item marketplace (Gold Bars) |

## Getting Help

- Open the Guides menu to browse documentation at any time.
- Join the Discord server via the **DISCORD** button in the top-right player menu.

## Opening Module Menus

Each module adds a button to the top-right player menu and a key binding you can
rebind under *Options → Controls → MOD CONTROLS*:

| Module | Button | Default Key |
|---|---|---|
| AtlasBanking | **BANKING** | **N** |
| AtlasExchange | **EXCHANGE** | **J** |
| AtlasBuildSectors | **BUILD SECTOR** | **B** |

## Banking

If AtlasBanking is installed, open it with the **BANKING** button (or **N**) to
check your balance, deposit and withdraw credits, transfer credits to other
players, and view your transaction history.

## Exchange

If AtlasExchange is installed, open it with the **EXCHANGE** button (or **J**) to
buy and sell ships, stations, items, and weapons for Gold Bars. You earn 1–3
Gold Bars as a daily login reward.

## Build Sectors

If AtlasBuildSectors is installed, open the **BUILD SECTOR** menu (or **B**) to
warp into a private creative sector, protected from other players and hostile
NPCs. You can invite friends, manage permissions, and spawn blueprints there.
